### 页面说明
> 德育管理分为以下两个页面德育班级管理和德育个人管理，命名为：班级综评，学生综评;

 > 班级综评分为班级综合分数排名页面和德育班级项目页面，班级综合分数排名页面可以按学期，年级和班级查询当前班级综合分数排名，还可以查看各班级的的评分列表，可以按班级搜索评分情况，可以对班级评分进行新增和删除,以及德育班级评分详情查询等功用;
 
 > 德育班级项目页面具有德育班级项目列表查询，新增，修改，删除以及停用和启用等功能;
 
 > 个人综评分为个人综合分数排名页面和德育个人项目页面，个人综合分数排名页面可以按学期，年级和班级查询当前个人综合分数排名，还可以查看每个人的评分列表，可以按个人搜索评分情况，可以对个人评分进行新增和删除,以及德育个人评分详情查询等功用;
 
 > 德育个人项目页面具有德育个人项目列表查询，新增，修改，删除以及停用和启用等功能;
 
 >声明开发测试时前端需要在url上加上test=true参数，标示为测试接口，接口会返回静态数据以提供给前端调试页面使用，前端需要使用测试接口完成数据渲染工作后，等到正式接口开发完成后，才能调用正式接口.

#### 12.1.2 德育班级评分列表查询接口
> 德育班级评分列表查询接口当前标签下的班级列表数据，当学期id为空时，返回全部班级列表数据。本接口支持分页查询班级评分列表

> 需要使用Token验证查询者身份

``` json
Url:        /qxy/moraleducation/class?termId=[termId]&page=[page]&size=[size]&token=[token]
Method:     GET
Header:     Content-type:application/json
Parameter:
    - `termId`:    学期ID，为空返回全部所有学期,默认为0
    - `page` ：     page对应的分页编号数字，当page＝0或者page为空时，返回第一页
    - `size` ：     size对应的每页数据条数，当size＝0或者size为空时，默认一页返回12条
    - `token` ：      token
Response:
    - `status`：     0->token is OK 
                    11->token is invalid
                    12->token is timeout
                    13->Illegal interface calls, not power use interface
    - `classScoreList`:    年级班级分值列表
    - `count`:          班级列表总数
    - `page`:           当前的分页编号
    - `totalScore`:     年级班级综合分值
    - `ranking`:        排名
    - `cid`:            班级id
```
> **返回结果示例：**
```
{
    "status": 0,
    "data": {
        "page":"0",   #为0时返回的全部分页
        "size":"14",   #为0时默认返回每页12条
        "count":"3",
        "classScoreList": [
						{
						"cid": "4",
						"gradeClass": "初一1班",
						"totalScore": "90",
						"ranking": "12",
						"updateTime":"2016-09-19"
						},
						{
						"cid": "4",
						"gradeClass": "初一2班",
						"totalScore": "92",
						"ranking": "11",
						"updateTime":"2016-09-19"
						},
						{
						"cid": "4",
						"gradeClass": "初一3班",
						"totalScore": "95",
```

#### 12.1.3 德育班级评分班级搜索接口
> 需要使用Token验证查询者身份
>根据班级cid进行搜索，不传参数或为空的时候，返回全部班级的分数列表。

``` json
Url:        /qxy/moraleducation/class/search?cid=[cid]&token=[token]
Method:     GET
Header:     Content-type:application/json
Parameter:
    - 'cid':        班级id
    - `token` ：      token
Response:
    - `status`：0->OK 
                1->要查找的班级不存在
                2->无权限查找
                11->token is invalid
                12->token is timeout
                13->Illegal interface calls, not power use interface
```
> **返回结果示例：**
```
{
    "status": 0,
    "data": {
           "cid": "4",
           "gradeClass": "初一3班",
           "totalScore": "95",
           "ranking": "9"
    }
}
```
