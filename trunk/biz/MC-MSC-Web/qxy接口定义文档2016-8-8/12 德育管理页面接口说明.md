### 页面说明
 >声明开发测试时前端需要在url上加上test=true参数，标示为测试接口，接口会返回静态数据以提供给前端调试页面使用，前端需要使用测试接口完成数据渲染工作后，等到正式接口开发完成后，才能调用正式接口.

> 德育管理分为以下两个页面德育班级管理和德育个人管理，命名为：班级综评，学生综评;

 > 班级综评分为班级综合分数排名页面和德育班级项目页面，班级综合分数排名页面可以按学期，年级和班级查询当前班级综合分数排名，还可以查看各班级的的评分列表，可以按班级搜索评分情况，可以对班级评分进行新增和删除,以及德育班级评分详情查询等功用;
 
 > 德育班级项目页面具有德育班级项目列表查询，新增，修改，删除以及停用和启用等功能;
 
 > 个人综评分为个人综合分数排名页面和德育个人项目页面，个人综合分数排名页面可以按学期，年级和班级查询当前个人综合分数排名，还可以查看每个人的评分列表，可以按个人搜索评分情况，可以对个人评分进行新增和删除,以及德育个人评分详情查询等功用;
 
 > 德育个人项目页面具有德育个人项目列表查询，新增，修改，删除以及停用和启用等功能;

#### 12.1.2 德育班级评分列表查询接口
> 德育班级评分列表查询接口当学期id为空时，返回默认学期下班级列表数据。本接口支持分页查询班级评分列表

> 需要使用Token验证查询者身份

``` json
Url:        /qxy/moraleducation/class?termId=[termId]&order=[order]&page=[page]&size=[size]&token=[token]
Method:     GET
Header:     Content-type:application/json
Parameter:
    - `termId`:    学期ID，如果不传或者为0，则查寻默认学期的评分班级列表；
    - `page` ：     page对应的分页编号数字，当page＝0或者page为空时，返回第一页
    - `size` ：     size对应的每页数据条数，当size＝0或者size为空时，默认一页返回12条
    - 'order':       结果排序（按照分值排序）：为空或=0-正序排列，1-倒叙排列
    - `token` ：      token
Response:
    - `status`：     0->token is OK 
                    11->token is invalid
                    12->token is timeout
                    13->Illegal interface calls, not power use interface
    - `page`:              当前的分页编号
    - `size`:              当前的每页数据量
    - `count`:             班级列表总数
    - `classScoreList`:    年级班级分值列表    
    - `cid`:               班级id
    - `className`:         班级名称（年级名+班级名）
    - `totalScore`:        年级班级综合分值
    - `ranking`:           排名
    - `updateTime`:        更新时间

```
> **返回结果示例：**
```
{
    "status": 0,
    "data": {
        "page":"1",   
        "size":"14",   
        "count":"3",
        "classScoreList": [
			{
				"cid": "4",
				"className": "初一1班",
				"totalScore": "90",
				"ranking": "12",
				"updateTime":"2016-09-19 12:00"
			},
			{
				"cid": "4",
				"className": "初一2班",
				"totalScore": "92",
```
